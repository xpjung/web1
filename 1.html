HTML이란 무엇인가?

Hypertext Markup Language(HTML) is the standard markup language for creating web pages and web applications. browsers receive HTML documents from a web server of from local storage and render them into multimedia web pages.

HTML describes the structure of a web page semantically and originally included cues for the apperence of the document. HTML elements are the building blocks of HTML pages.
